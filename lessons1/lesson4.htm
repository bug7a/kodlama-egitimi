Matematiksel işlemler

Bir algoritma içerisinde, dört işlem işaretlerini kullanarak; bilgisayardan, aritmetik işlemler yapmasını isteyebiliriz.

Aşağıdaki örnek kod ile bilgisayardan, bizim belirlediğimiz bir matematiksel işlemi yapmasını ve sonucunu, ekrana yazdırmasını isteyelim.

Yukarıdaki örnekte; toplama işleminin (2 + 1) sonucu, toplam isimli değişkene aktarılmıştır.

Şimdi ise, içinde sayı tutan bir değişken ile matematiksel işlemler yapalım ve işlem sonucunu, ekrana yazdıralım.

Bilmemiz gereken bazı kurallar

Toplama + işaretini, sayı veya içine sayı atanmış değişkenler ile kullandığımızda; toplama işlemi yapar.

Bir matematiksel işlemin, öncelikli olarak yapılmasını istediğimizde; onu, parantez içinde ( ... ) yazmalıyız.

Kod içerisinde, üzerlerinde matematiksel işlemler yapabileceğimiz birçok sayısal bilgiye ulaşabiliriz.

Aşağıdaki örnekte; robotun, yolun sonuna gitmesi için kaç birim ilerlemesi gerektiğini, robot.konum ve zemin.sonKonum değişkenlerini kullanarak hesaplamaya çalışalım.


robot.konum
robotun, zemin üzerindeki konumunu taşıyan değişken.
zemin.sonKonum
zeminin uzunluğunu taşıyan değişken.

zemin.sonKonum değişkeninden (zeminin uzunluğundan) robot.konum değişkenini (robotun konumunu) çıkardığımızda, robotun son konuma olan uzaklığını hesaplamış oluyoruz.

DEĞİŞKENLERİN KULLANIMI
DÖNGÜLER
// İşlem sonucunu tutmak için bir değişken tanımlayalım. var toplam // Toplama işlemin yapalım. toplam = 2 + 1 // Değişkenimizin içeriğini ekrana yazdıralım. yaz("İşlem sonucu: " + toplam)
// İşlem sonucunu tutmak için bir değişken tanımlayalım. var sonuc // Başka bir değişken tanımlayalım ve içine bir sayı aktaralım. var sayi1 = 2 // Matematik işleminin sonucunu değişkenimizin içine aktaralım. sonuc = (sayi1 + 3) * sayi1 // Değişkenimizin içeriğini ekrana yazdıralım. yaz("İşlem sonucu: " + sonuc)
// Kullanılacak değişkenlerin içeriklerini ekranda göster. yaz("Başlangıç konumum: " + robot.konum) yaz("Zemin son konum: " + zemin.sonKonum) // Robotun hedefe olan uzaklığını hesapla. var mesafe = zemin.sonKonum - robot.konum // Bir satır boşluk bırak. yaz("") // Sonucu ekranda göster. yaz("Hedefe ulaşmak için " + mesafe + " kere ilerlemem gerekiyor.")
