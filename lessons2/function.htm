KONULAR

Fonksiyon oluşturmak (function)

Fonksiyonlar, bir kod grubuna, isim vererek; onu, ismi ile istediğimiz yerlerde (kod satırlarında) çalıştırabilmemizi sağlayan yapılardır.

Aşağıdaki örneği inceleyelim:

selamVer() isminde bir fonksiyon oluşturalım. Bu fonksiyon, çalıştırıldığında ekrana "Merhaba Dünya!" cümlesini yazdırsın.

Fonksiyonun, parametre ile çalışması

Oluşturduğumuz fonksiyonun, çalışırken kullanması için ona bilgiler (parametreler) gönderebiliriz.

Şimdi, bir örnek yapalım:

ikiKereYaz(mesaj) isminde bir fonksiyon oluşturalım. Ona, parametre olarak gönderdiğimiz bilgiyi ekrana iki kere yazdırsın.

Bilmemiz gereken bir kural

Bir değişken veya fonksiyon ismi, sayı ile başlayamaz.

mesaj isimli parametrenin içeriği, fonksiyon çalıştırılırken aktarılır. Ve sadece, fonksiyona ait süslü parantezin içerisinde kullanılabilir.

Şimdi, başka bir örnek yapalım:

carpimTablosuOlustur(sayi) ismindeki fonksiyonumuz; ona gönderilen sayının, 1'den 5’e kadar bütün sayılar ile çarpımını ekrana yazdırsın.

Fonksiyonun, cevap vermesi

Yaptığı işlemler sonucunda elde ettiği bilgiyi, return kelimesini kullanarak, fonksiyonun çalıştırıldığı kod satırına geri gönderebiliriz.

Şimdi, bir örnek yapalım:

Üç sayının ortalamasını hesaplayan ve cevabı geri gönderen bir fonksiyon oluşturalım.

cevap değişkeni, geçici kullanım için tanımlanmış olup, tanımlandığı süslü parentezin dışından ulaşılamaz.

// Fonksiyonu oluştur. var ikiKereYaz = function(mesaj) { yaz(mesaj) yaz(mesaj) } // Fonksiyonu bir parametre ile çalıştır. ikiKereYaz("Merhaba Dünya!")
// Fonksiyon oluştur. var carpimTablosuOlustur = function(sayi) { // 5 kere tekrar et. for (var i = 1; i <= 5; i++) { yaz(sayi + " x " + i + " = " + (sayi * i)) } } // Fonksiyonu 3 kere, farklı parametreler ile çalıştır. carpimTablosuOlustur(3) carpimTablosuOlustur(7) carpimTablosuOlustur(9)
