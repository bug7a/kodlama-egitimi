KONULAR

Düğme (Button)

Kullanıcının, bir alana basarak; belirlenen fonksiyonu çalıştırabilmesini sağlayan nesne.

Şimdi, bir örnek yapalım:

Sayfa rengini, rasgele olarak değiştiren bir düğme oluşturalım.

page nesnesi, kullanılabilen ekranın renk (.color) ve yükseklik (.height) gibi özelliklerinin yönetiminden sorumludur.

METİN KUTUSU NESNESİ
RESİM NESNESİ
// Nesneyi taşıyacak değişkeni tanımla. var dugme // İlk çalıştırılan özel fonksiyon. var start = function() { // Bir düğme nesnesi oluştur ve değişkene aktar. dugme = createButton() // Düğme metnini belirle. dugme.text = "Renk Değiştir" // Sola olan mesafeyi belirle. dugme.left = 50 // Yukarıya olan mesafeyi belirle. dugme.top = 50 // Nesnenin genişliğini belirle. dugme.width = 180 // Nesnenin rengini beyaz olarak belirler. dugme.color = "white" // Düğmeye basıldığında çalıştırılacak fonksiyonu belirle. dugme.onClick(renkDegistir) } // Sayfa rengini rasgele değiştiren fonksiyon. var renkDegistir = function() { // 1 den 3 e kadar rasgele bir sayı seç. var rasgeleNumara = random(1, 3) // Eğer değişken içeriği 1, 2 veya 3 ise, if (rasgeleNumara == 1) { // Sayfayı, portakal kırmızısı ile boya. page.color = "orangered" } else if (rasgeleNumara == 2) { // Sayfayı, deniz yeşili ile boya. page.color = "seagreen" } else { // Sayfayı, çelik mavisi ile boya. page.color = "steelblue" } // Düğmeyi, sayfa rengi ile aynı yap. dugme.color = page.color }
