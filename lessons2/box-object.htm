KONULAR

Kutu (Box)

Diğer nesneleri, bir alanda gruplamak veya ekranda bir şekil oluşturmak için kullanılabilen nesne.

Şimdi, bir örnek yapalım:

Sınırları renkli, içi şeffaf (arkası görünen) bir dikdörtgen oluşturalım.

dikdortgen isimli nesnenin özelliklerinde, değişiklikler yapın ve sonucu gözlemleyin.

RESİM NESNESİ
DİZİLERİN KULLANIMI
// Nesneyi taşıyacak değişkeni tanımla. var dikdortgen // İlk çalıştırılan özel fonksiyon. var start = function() { // Sayfayı beyaz ile boya. page.color = "white" // Bir kutu nesnesi oluştur. dikdortgen = createBox() // Nesnenin konumunu belirle. dikdortgen.left = 50 dikdortgen.top = 50 // Nesnenin boyutunu belirle. dikdortgen.width = 150 dikdortgen.height = 80 // Nesnenin rengini şeffaf olarak belirle. dikdortgen.color = "transparent" // Nesnenin kenarlık kalınlığını belirle. // (0: kenarlık yok) dikdortgen.border = 2 // Kenarlık rengi portakal kırmızısı olsun. dikdortgen.borderColor = "orangered" }
